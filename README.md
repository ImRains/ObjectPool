# Godot Object Pool (Godot 4.x)
一个专为 Godot 4 优化的高性能对象池插件，旨在减少高频对象（如子弹、特效、飘字）实例化带来的内存抖动和 CPU 开销。

## ✨ 特性
+ **双容器管理**：使用 `Array` 存储空闲对象，`Dictionary` 管理活跃对象，确保检索效率为 $O(1)$。
+ **动态衰减机制 (Decay)**：自动释放超出初始大小的闲置节点，防止内存长期占用。
+ **慢加载方案**：支持分帧创建实例，避免初始化大量对象时产生瞬时掉帧。
+ **安全保护**：具备最大容量限制（Max Pool Size）和实例有效性检查，防止空指针崩溃。
+ **状态自动化**：回收时自动禁用物理处理、隐藏显示，取出时自动恢复。

---

## 🛠 安装方法
1. 将 `addons/object_pool_plugin` 文件夹复制到你项目的 `res://addons/` 目录下。
2. 前往 **项目设置 -> 插件**，启用 **Object Pool**。
3. 在场景中添加 `ObjectPool` 节点。
